Node-Link Diagram

Circular Dendogram with Curved Edges

Interactivity

Hover to see the tooltip with name and amount of cases. Hover to any circle of last 2 layers to filter by the type of case.

Encoding

Each level shows a pacticular column from data. The first level (root) colored dark green shows the city (San Francicso in this case).
The second level colored light green shows the call type group of cases that happened in that city.
The third level colored light purple shows the call type and how many of these cases happened per call type group in the city.
The fourh and the last biggest lap colored dark purple shows call final disposition of hierarchical cases divided by each case type happened in that city.

Wrangling

Data was filtered to show only April 2020 cases and only San Francicso city. The column "Neighborhooods - Analysis Boundaries" was renamed to "Neighborhooods". Data in csv format can be found here.

Conclusion

Using this chart we can see the April hierarchy, how the call gets assigned to a call type group, then to a call type and then the disposition of the call.
For example, the calls that got assigned to "Water Rescue" were disposed as "Code 2 Transport" and "Electrical Hazard" calls were disposed as "Fire".
On the other hand, "Medical Incident" calls were disposed to 12 different categories.
